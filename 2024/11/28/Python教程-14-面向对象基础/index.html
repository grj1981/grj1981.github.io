14.1 面向对象概念

面向对象编程（OOP）是一种编程范式，将数据和操作封装在对象中。

核心概念

  • 类（Class）：对象的蓝图/模板
  • 对象（Object）：类的实例
  • 属性（Attribute）：对象的数据
  • 方法（Method）：对象的行为

为什么使用 OOP

  • 封装：隐藏内部实现
  • 复用：代码可重复使用
  • 扩展：易于添加新功能
  • 维护：模块化便于管理

14.10 本章小结

  • 类是对象的模板，对象是类的实例
  • __init__ 是构造方法，self 代表实例
  • 实例属性属于对象，类属性属于类
  • 实例方法、类方法（@classmethod）、静态方法（@staticmethod）
  • 使用 _ 和 __ 约定访问控制
  • @property 用于属性管理
  • __str__ 和 __repr__ 定义字符串表示
