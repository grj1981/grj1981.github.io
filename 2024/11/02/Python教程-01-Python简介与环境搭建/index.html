1.1 什么是 Python

Python 是一种高级、解释型、通用编程语言，由 Guido van Rossum 于 1991 年首次发布。它以简洁、易读的语法著称，被广泛应用于 Web 开发、数据分析、人工智能、自动化脚本、科学计算等领域。

Python 的核心优势

  • 简洁易读：语法接近自然语言，代码量少，学习曲线平缓
  • 跨平台：支持 Windows、macOS、Linux 等主流操作系统
  • 丰富的标准库：内置大量模块，覆盖文件操作、网络通信、数据处理等
  • 庞大的第三方生态：PyPI 拥有超过 40 万个包，满足各种开发需求
  • 多范式支持：支持面向对象、函数式、过程式编程
  • 强大的社区：全球开发者活跃贡献，文档完善，教程丰富

Python 的应用领域

领域 代表框架/库
Web 开发 Django, Flask, FastAPI
数据分析 Pandas, NumPy, Matplotlib
人工智能 TensorFlow, PyTorch, Scikit-learn
自动化运维 Ansible, Fabric, Paramiko
爬虫开发 Scrapy, Requests, BeautifulSoup
游戏开发 Pygame, Panda3D

1.2 Python 版本选择

Python 2 vs Python 3

  • Python 2 已于 2020 年 1 月 1 日正式停止维护
  • Python 3 是当前唯一推荐版本，持续更新中
  • 本教程基于 Python 3.10+ 编写

版本选择建议

用途 推荐版本
初学者学习 最新稳定版（如 3.12）
企业项目 根据项目依赖选择，建议 3.9+
数据科学/AI 3.8+（注意库兼容性）

1.3 安装 Python

Windows 安装

  1. 访问 Python 官网 下载最新 Windows 安装包
  2. 运行安装程序，务必勾选 “Add Python to PATH”
  3. 选择 “Install Now” 完成安装
  4. 打开命令提示符，输入 python --version 验证安装

1.4 开发环境配置

推荐编辑器

编辑器 特点 适用场景

VS Code 配置

  1. 安装 Python 扩展
  2. 打开命令面板（Ctrl+Shift+P），选择 Python: Select Interpreter
  3. 选择已安装的 Python 解释器
  4. 创建 .py 文件即可开始编写

1.6 Python 代码规范

PEP 8 基础规范

  • 缩进：使用 4 个空格（不要用 Tab）
  • 行长度：每行不超过 79 个字符
  • 命名：
    • 变量/函数：snake_case（如 user_name）
    • 类名：PascalCase（如 UserProfile）
    • 常量：UPPER_CASE（如 MAX_SIZE）
  • 注释：使用 # 单行注释，""" """ 多行文档字符串

1.7 本章小结

  • Python 是一门简洁、强大、应用广泛的高级编程语言
  • 始终使用 Python 3 版本进行学习
  • 安装时记得勾选 “Add Python to PATH”
  • VS Code + Python 扩展是轻量高效的开发组合
  • 遵循 PEP 8 规范编写代码

练习题

  1. 在你的系统上安装 Python 3.10+ 版本
  2. 安装 VS Code 并配置 Python 扩展
  3. 创建 hello.py 文件，输出你的姓名和一句自我介绍
  4. 在交互式解释器中尝试以下运算：
    • 10 + 20
    • 100 - 37
    • 6 * 7
    • 15 / 3
