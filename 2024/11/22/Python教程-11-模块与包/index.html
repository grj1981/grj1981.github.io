11.1 模块的概念

模块是包含 Python 代码的 .py 文件，用于组织和管理代码。

为什么使用模块

  • 代码复用：在多个程序中共享代码
  • 命名空间：避免名称冲突
  • 可维护性：将代码分解为可管理的部分
