дёҠдёҖзҜҮгҖҠHexo еҚҡе®ўдёүйЎ№еҹәзЎҖдҝ®еӨҚгҖӢи§ЈеҶідәҶ RSSгҖҒSitemapгҖҒиҜ„и®әзі»з»ҹзҡ„й«ҳдјҳе…Ҳзә§зЎ¬жҖ§й—®йўҳпјҢеҚҡе®ўе…·еӨҮдәҶзЁіеӣәзҡ„еҹәзЎҖи®ҫж–ҪгҖӮдҪҶ Lighthouse е®ЎжҹҘиҝҳжҡҙйңІдәҶжӣҙеӨҡй—®йўҳпјҡ

дёҖгҖҒSEO дјҳеҢ–

1.1 Open Graph / Twitter Card

й—®йўҳ: ж–Үз« еҲҶдә«еҲ°еҫ®дҝЎгҖҒTwitter ж—¶жІЎжңүж ҮйўҳгҖҒжҸҸиҝ°гҖҒзј©з•ҘеӣҫпјҢеҸӘжңүе…үз§ғз§ғзҡ„ URLгҖӮ

ж–№жЎҲ: NexT дё»йўҳе·ІйӣҶжҲҗ Hexo еҶ…зҪ®зҡ„ open_graph() ж–№жі•пјҢеңЁ head-unique.swig дёӯи°ғз”Ёпјҡ

е®ғиҮӘеҠЁд»ҺйЎөйқў front-matter жҸҗеҸ– titleгҖҒdescriptionгҖҒimage з”ҹжҲҗ OG ж Үзӯҫпјҡ

ж— йңҖжүӢеҠЁй…ҚзҪ®гҖӮеҰӮжһңйңҖиҰҒиҮӘе®ҡд№үпјҢHexo ж”ҜжҢҒеңЁ _config.yml дёӯдј еҸӮпјҡ

1.2 JSON-LD з»“жһ„еҢ–ж•°жҚ®

й—®йўҳ: жҗңзҙўеј•ж“ҺдёҚзҗҶи§ЈйЎөйқўзҡ„з»“жһ„е’ҢеҶ…е®№зұ»еһӢпјҢж— жі•з”ҹжҲҗ Rich SnippetпјҲеҜҢж‘ҳиҰҒпјүгҖӮж–Үз« еңЁжҗңзҙўз»“жһңйЎөеҸӘиғҪжҳҫзӨәж Үйўҳе’Ңй“ҫжҺҘпјҢжІЎжңүдҪңиҖ…гҖҒеҸ‘еёғж—¶й—ҙзӯүеўһејәдҝЎжҒҜгҖӮ

ж–№жЎҲ: зј–еҶҷдәҶдёҖдёӘ json-ld.js helperпјҢеңЁ head-unique.swig дёӯи°ғз”Ё {{ json_ld() }}пјҢж №жҚ®йЎөйқўзұ»еһӢиҫ“еҮәдёүз»„з»“жһ„еҢ–ж•°жҚ®пјҡ

е®ҢжҲҗеҗҺз”Ё Google зҡ„ Rich Results Test йӘҢиҜҒпјҢзЎ®и®Ө BlogPostingгҖҒBreadcrumbList з»“жһ„жӯЈзЎ®гҖӮ

1.3 з»ҹи®Ўзі»з»ҹжҺҘе…Ҙ

й—®йўҳ: еҚҡе®ўдёҠзәҝеҗҺжІЎжңүд»»дҪ•и®ҝй—®ж•°жҚ®пјҢж— жі•дәҶи§ЈиҜ»иҖ…жқҘжәҗгҖҒзғӯй—ЁеҶ…е®№гҖӮ

ж–№жЎҲ: NexT дё»йўҳеҶ…зҪ®дәҶ Google Analytics е’ҢзҷҫеәҰз»ҹи®Ўзҡ„жіЁе…ҘжЁЎжқҝпјҢеҸӘйңҖй…ҚзҪ® tracking_idпјҡ

й…ҚзҪ®еҗҺиҮӘеҠЁеңЁ <head> дёӯжіЁе…Ҙ gtag.js и„ҡжң¬гҖӮonly_pageview: false иЎЁзӨәеҠ иҪҪе®Ңж•ҙ gtagпјҢж”ҜжҢҒдәӢд»¶иҝҪиёӘе’Ңз”ЁжҲ·еҲҶжһҗгҖӮеҰӮжһңеҸӘжғіз»ҹи®Ў PVпјҢи®ҫдёә true дјҡж”№з”Ё sendBeacon иҪ»йҮҸжҺЁйҖҒгҖӮ

1.4 иҮӘеҠЁжҸҸиҝ°з”ҹжҲҗ

й—®йўҳ: 91 зҜҮж–Үз« пјҢжүӢеҠЁеҶҷ description еӨӘиҖ—ж—¶гҖӮжІЎжңү description зҡ„йЎөйқўеңЁжҗңзҙўеј•ж“Һдёӯж‘ҳиҰҒдёәз©әпјҢеҪұе“ҚзӮ№еҮ»зҺҮгҖӮ

ж–№жЎҲ: еҶҷдәҶдёҖдёӘ auto-description.js иҝҮж»ӨеҷЁпјҢеңЁ after_post_render йҳ¶ж®өиҮӘеҠЁжҸҗеҸ–еүҚ 200 еӯ—з¬Ұпјҡ

и§„еҲҷпјҡ

  • е·ІжүӢеҠЁеЎ«еҶҷ description зҡ„ж–Үз« дёҚиҰҶзӣ–
  • еҺ»жҺү HTML ж Үзӯҫе’Ңд»Јз Ғеқ—
  • еҺ»жҺү Markdown иҜӯжі•з¬ҰеҸ·
  • жҲӘеҸ–еүҚ 200 еӯ—з¬ҰпјҢи¶…еҮәеҠ  ...

иҝҷж ·жүҖжңүйЎөйқўйғҪиҮӘеҠЁжңүдәҶ SEO еҸӢеҘҪзҡ„е…ғжҸҸиҝ°гҖӮ

дәҢгҖҒз”ЁжҲ·дҪ“йӘҢдјҳеҢ–

2.1 иҮӘе®ҡд№ү 404 йЎөйқў

й—®йўҳ: GitHub Pages й»ҳи®Ө 404 йЎөйқўеӨӘз®ҖйҷӢпјҢи®ҝе®ўйҒҮеҲ°жӯ»й“ҫзӣҙжҺҘзҰ»ејҖгҖӮ

ж–№жЎҲ: еҲӣе»ә source/404.mdпјҢдҪҝз”Ё NexT йЎөйқўжЁЎжқҝжёІжҹ“пјҡ

еҠ дёҠе“ҒзүҢиүІж ·ејҸзҡ„жҢүй’®пјҲ#37c6c0пјүпјҢhover ж—¶дёҠжө®пјҡ

2.2 йҳ…иҜ»ж—¶й—ҙдј°з®—

й—®йўҳ: иҜ»иҖ…жү“ејҖж–Үз« ж—¶дёҚзҹҘйҒ“еӨ§жҰӮйңҖиҰҒеӨҡд№…иғҪиҜ»е®ҢпјҢе®№жҳ“жІЎжңүеҝғзҗҶйў„жңҹгҖӮ

ж–№жЎҲ: е®үиЈ… hexo-symbols-count-time жҸ’д»¶пјҢиҮӘеҠЁи®Ўз®—еӯ—ж•°е’Ңйў„дј°йҳ…иҜ»ж—¶й—ҙпјҡ

дё»йўҳй…ҚзҪ®дёӯеҗҜз”Ёпјҡ

ж•Ҳжһңпјҡж–Үз« ж ҮйўҳдёӢж–№жҳҫзӨә рҹ“„ еӯ—ж•°: 2,350 / рҹ•җ йҳ…иҜ»ж—¶й—ҙ вүҲ 12 еҲҶй’ҹгҖӮеә•йғЁе…Ёз«ҷз»ҹи®ЎжҳҫзӨәеҚҡе®ўжҖ»еӯ—ж•°гҖӮ

2.3 зӣёе…іж–Үз« жҺЁиҚҗ

й—®йўҳ: иҜ»иҖ…зңӢе®ҢдёҖзҜҮж–Үз« еҗҺпјҢдёҚзҹҘйҒ“иҝҳжңүд»Җд№Ҳзӣёе…іеҶ…е®№еҸҜд»ҘзңӢпјҢзјәе°‘еҒңз•ҷеј•еҜјгҖӮ

ж–№жЎҲ: еңЁ after_post_render иҝҮж»ӨеҷЁдёӯе®һзҺ°ж ҮзӯҫеҢ№й…Қз®—жі•пјҡ

еҚЎзүҮеұ•зӨәж•ҲжһңпјҡдҪҝз”Ё CSS Grid иҮӘйҖӮеә”еёғеұҖпјҢжӮ¬еҒңж—¶иҫ№жЎҶеҸҳдёәе“ҒзүҢиүІе№¶иҪ»еҫ®дёҠжө®гҖӮ

2.4 ж–Үз« зі»еҲ—еҜјиҲӘ

й—®йўҳ: еғҸгҖҢC# еӯҰд№ з¬”и®°гҖҚиҝҷз§Қзі»еҲ—ж–Үз« жңү 22 зҜҮпјҢиҜ»иҖ…дёҚзҹҘйҒ“еҪ“еүҚиҜ»еҲ°з¬¬еҮ зҜҮгҖҒеүҚеҗҺжҳҜд»Җд№ҲгҖӮ

ж–№жЎҲ: йҖҡиҝҮиҮӘе®ҡд№үж Үзӯҫ series:xxx ж Үи®°еҗҢзі»еҲ—ж–Үз« пјҢжҢүж—ҘжңҹжҺ’еәҸпјҢиҮӘеҠЁз”ҹжҲҗеҜјиҲӘпјҡ

иҝҮж»ӨеҷЁе®һзҺ°пјҡ

иё©еқ‘и®°еҪ•пјҡdata.tags еңЁ after_post_render йҳ¶ж®өжҳҜ _Query еҜ№иұЎпјҢдёҚжҳҜжҷ®йҖҡж•°з»„пјҢArray.from() дјҡиҝ”еӣһе…Ё undefinedгҖӮеҝ…йЎ»дҪҝз”Ё forEach() еҺҹз”ҹиҝӯд»ЈгҖӮеҸҰеӨ–пјҢз”Ё TagModel.findOne({name: tagName}) жҹҘиҜўеҗ«еҶ’еҸ·зҡ„ tag name дјҡеӨұиҙҘпјҢйңҖиҰҒзӣҙжҺҘз”Ё tag еҜ№иұЎдёҠжҗәеёҰзҡ„ .posts еұһжҖ§гҖӮ

2.5 зҪ®йЎ¶еҠҹиғҪ

й—®йўҳ: йҰ–йЎөжҢүж—¶й—ҙеҖ’еәҸжҳҫзӨәж–Үз« пјҢйҮҚиҰҒе…¬е‘Ҡж— жі•зҪ®йЎ¶гҖӮ

ж–№жЎҲ: еүҚзҪ®иҝҮж»ӨеҷЁе’ҢеҗҺзҪ®жёІжҹ“еҷЁй…ҚеҗҲпјҡ

еңЁж–Үз«  front-matter дёӯеҠ дёҖиЎҢеҚіеҸҜзҪ®йЎ¶пјҡ

2.6 ж Үзӯҫдә‘ / еҲҶзұ»еҸҜи§ҶеҢ–

й—®йўҳ: /tags/ йЎөйқўеҸӘжңүеҲ—иЎЁпјҢзңӢдёҚеҮәе“Әдәӣж Үзӯҫж–Үз« еӨҡгҖӮ

ж–№жЎҲ: NexT дё»йўҳе·ІдҪҝз”Ё Hexo еҶ…зҪ®зҡ„ tagcloud() ж–№жі•пјҢжҢүж–Үз« ж•°иҮӘеҠЁи°ғж•ҙеӯ—еҸ·е’ҢйўңиүІпјҡ

жёІжҹ“д»Јз ҒеңЁ page.swig дёӯпјҡ

еҲҶзұ»йЎөйқўеҗҢзҗҶпјҢдҪҝз”Ё list_categories()гҖӮ

2.7 ж»ҡеҠЁеҠЁз”»

й—®йўҳ: йЎөйқўеҲҮжҚўе’ҢеҶ…е®№еҮәзҺ°иҝҮдәҺз”ҹзЎ¬гҖӮ

ж–№жЎҲ: NexT дё»йўҳеҶ…зҪ® Velocity.js еҠЁз”»еј•ж“ҺпјҢејҖз®ұеҚіз”Ёпјҡ

ж•ҲжһңпјҡйЎөйқўеҠ иҪҪж—¶ж–Үз« еқ—ж·Ўе…ҘгҖҒдҫ§иҫ№ж Ҹж»‘е…ҘгҖӮasync: false зЎ®дҝқеҠЁз”»жҢүеәҸж’ӯж”ҫгҖӮ

дёүгҖҒе“ҒзүҢи§Ҷи§үйҮҚжһ„

3.1 е“ҒзүҢиүІзі»з»ҹ

й—®йўҳ: еҚҡе®ўйўңиүІжқҘиҮӘ NexT й»ҳи®ӨзҒ°и“қпјҢеҲ°еӨ„йғҪжҳҜж ҮеҮҶ Bootstrap йЈҺж јпјҢжІЎжңүиҫЁиҜҶеәҰгҖӮ

ж–№жЎҲ: еңЁ source/_data/variables.styl дёӯе®ҡд№үе“ҒзүҢиүІдҪ“зі»пјҢиҰҶзӣ– NexT еҶ…зҪ®еҸҳйҮҸпјҡ

иҝҷеҘ—еҸҳйҮҸйҖҡиҝҮ NexT зҡ„ custom_file_path жіЁе…ҘжңәеҲ¶иҮӘеҠЁз”ҹж•ҲгҖӮиҰҶзӣ–еҺҹзҗҶпјҡsource/_data/variables.styl еңЁдё»йўҳ Stylus зј–иҜ‘ејҖе§Ӣж—¶еҠ иҪҪпјҢеҗҺз»ӯжүҖжңүеј•з”Ё $blueгҖҒ$link-color зҡ„ең°ж–№йғҪдјҡи§ЈжһҗдёәжҲ‘д»¬зҡ„е“ҒзүҢиүІгҖӮ

е®һйҷ…еҪұе“Қпјҡ

  • д»Јз Ғй«ҳдә®зҡ„и“қиүІ вҶ’ йқ’и“қ #37c6c0
  • ж–Үз« й“ҫжҺҘиүІ вҶ’ #4d96ffпјҢhover вҶ’ #ff6b6b
  • жҢүй’®иҫ№жЎҶе’Ңж–Үеӯ— вҶ’ е“ҒзүҢиүІ
  • е…ЁеұҖеңҶи§’ вҶ’ 20pxпјҲеҺҹжң¬жҳҜзӣҙи§’пјү

иё©еқ‘и®°еҪ•пјҡе“ҒзүҢиүІиҝҳеңЁ styles.styl дёӯиў«еӨҡеӨ„зӣҙжҺҘеј•з”ЁпјҲжҜ”еҰӮ 404 йЎөйқўзҡ„ .error-codeгҖҒзҪ®йЎ¶ж ҮиҜҶ .post-pinned-noticeгҖҒзі»еҲ—еҜјиҲӘ .series-nav зҡ„иҫ№жЎҶпјүпјҢиҝҷдәӣдёҚдјҡиҮӘеҠЁиў«еҸҳйҮҸиҰҶзӣ–пјҢйңҖиҰҒжүӢеҠЁжӣҙж–°гҖӮе»әи®®ж–°ж ·ејҸз»ҹдёҖдҪҝз”ЁеҸҳйҮҸпјҢе·Іжңүж ·ејҸйҖҗжӯҘиҝҒз§»гҖӮ

3.2 Hero еҢәеҹҹ

й—®йўҳ: йҰ–йЎөжү“ејҖзӣҙжҺҘжҳҜж–Үз« еҲ—иЎЁпјҢжІЎжңүе“ҒзүҢеұ•зӨәеҢәгҖӮж–°и®ҝе®ўж— жі•еҝ«йҖҹдәҶи§ЈвҖқиҝҷдёӘеҚҡе®ўжҳҜеҒҡд»Җд№Ҳзҡ„вҖқгҖӮ

ж–№жЎҲ: еңЁйҰ–йЎөйЎ¶йғЁеўһеҠ  Hero еұ•зӨәеҢәпјҢеҢ…еҗ«еӣӣдёӘйғЁеҲҶпјҡ

в‘  жЁЎжқҝ (themes/next/layout/_partials/hero.swig)пјҡ

в‘Ў еј•з”Ё (themes/next/layout/index.swig)пјҡ

в‘ў жү“еӯ—жңәеҠЁз”» (source/_data/body-end.swig)пјҡ

ж ёеҝғжңәеҲ¶пјҡеӣӣдёӘж ҮиҜӯеҫӘзҺҜеҲҮжҚўпјҢе…ҲйҖҗеӯ—жү“еҮә вҶ’ еҒңз•ҷ 2 з§’ вҶ’ йҖҗеӯ—еҲ йҷӨ вҶ’ еҲҮжҚўеҲ°дёӢдёҖдёӘгҖӮжү“еӯ—йҖҹеәҰ 100ms/еӯ—пјҢеҲ йҷӨйҖҹеәҰ 50ms/еӯ—пјҢе…үж Ү 0.8s й—ӘзғҒгҖӮ
