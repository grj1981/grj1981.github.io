2.5 各类型详解

nil（空）

nil 表示”没有有效值”，用于：

  • 变量未赋值时默认值为 nil
  • 删除表的元素：t.key = nil
  • 表示逻辑假（在条件判断中 nil 等同于 false）

boolean（布尔）

只有 true 和 false 两个值。注意：在 Lua 中，nil 和 false 为假，其他所有值都为真（包括 0 和空字符串）。
