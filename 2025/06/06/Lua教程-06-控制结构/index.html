6.8 本章小结

结构 语法 说明
if if 条件 then ... end 条件分支
while while 条件 do ... end 先判断后执行
repeat repeat ... until 条件 先执行后判断
for 数值 for i=a,b,step do ... end 固定次数循环
for 泛型 for k,v in iter do ... end 遍历迭代器
break break 跳出当前循环
goto goto label 跳转到标签

练习题

  1. 用 for 循环打印斐波那契数列的前 20 个数
  2. 用 while 循环实现一个”猜数字”游戏，统计用户猜的次数
  3. 用 repeat...until 实现一个密码输入验证，直到输入正确为止
  4. 写一个函数 is_prime(n) 判断 n 是否为质数
  5. 在 1~100 中找到所有水仙花数（三位数各位立方和等于本身，如 153）
  6. 用 goto 模拟 continue 打印 1~30 中不是 3 或 5 的倍数的数
