7.8 本章小结

  • 函数用 function 定义，可以具名或匿名
  • 参数不足时为 nil，多余参数被忽略
  • 支持多返回值，用逗号分隔
  • 变长参数 ... 配合 {...} 或 select() 使用
  • 可以用 table 模拟具名参数
  • 函数是第一类值，可作为参数和返回值
