Lua 简介与环境搭建

发表于 2025-06-01 分类于 教程 ， Lua Waline：
Lua 脚本语言入门第一课，介绍 Lua 轻量高效的特性与应用场景（游戏开发、嵌入式）、开发环境搭建、常用 IDE（ZeroBrane Studio / VS Code）配置，编写第一个 Lua 脚本。

1.1 什么是 Lua

Lua 是一种轻量、小巧、可扩展的脚本语言，由巴西里约热内卢天主教大学（PUC-Rio）的 Roberto Ierusalimschy、Waldemar Celes 和 Luiz Henrique de Figueiredo 于 1993 年开发。名字 “Lua” 在葡萄牙语中意为 “月亮”。

Lua 的设计哲学可以概括为三个词：轻量、可嵌入、高效。它不是一个用来编写独立大型应用的”大语言”，而是一个胶水语言——被嵌入到其他宿主程序中，为宿主提供灵活的脚本能力。

Lua 的核心特性

  • 简洁优雅：语法简单，语义清晰，仅有约 20 个关键字
  • 动态类型：变量无类型声明，运行时确定类型
  • 自动内存管理：基于标记-清除（Mark-and-Sweep）的垃圾回收
  • 一等函数：函数是第一类值，可赋值给变量、作为参数传递、作为返回值
  • 表（Table）：Lua 唯一的数据结构容器，同时也是数组、字典、模块、类等的统一实现
  • 协程（Coroutine）：轻量级并发支持
  • 可嵌入性：C API 完善，可轻松集成到 C/C++ 程序中

Lua 的应用场景

领域 代表案例
游戏开发 Unity（XLua/ToLua）、World of Warcraft（插件）、Roblox、Corona SDK
嵌入式系统 路由器固件、物联网设备脚本
Web 服务 OpenResty（Nginx + Lua）、Redis 脚本（EVAL）
配置文件 Nginx、Wireshark、VLC、I3 窗口管理器
工具链 Neovim 插件（VimScript 替代者）、Awesome WM

Lua 的优势在于极致的小巧和嵌入性，如果你的应用需要一个脚本引擎来扩展功能，Lua 是最佳选择之一。

1.6 本章小结

  • Lua 是一种轻量、高效、可嵌入的脚本语言
  • 擅长作为胶水语言扩展宿主程序功能
  • 安装方式简单，一行命令即可
  • 交互式模式适合快速验证语法
  • 脚本模式用于编写完整程序
