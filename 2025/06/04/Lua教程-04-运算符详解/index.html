4.2 关系运算符

运算符 含义 示例 结果
== 等于 5 == 5 true
~= 不等于 5 ~= 3 true
< 小于 3 < 5 true
> 大于 5 > 3 true
<= 小于等于 5 <= 5 true
>= 大于等于 5 >= 3 true

比较规则

记住：Lua 中 == 用于相等判断，~= 用于不等判断。没有 ===（严格等于），因为不会发生类型转换。

4.3 逻辑运算符

运算符 含义 说明
and 逻辑与 短路求值
or 逻辑或 短路求值
not 逻辑非 取反

短路求值机制

and： 如果第一个操作数为假，返回第一个操作数；否则返回第二个操作数。

or： 如果第一个操作数为真，返回第一个操作数；否则返回第二个操作数。

4.6 运算符优先级

从高到低：

优先级 运算符
1（最高） ^
2 -（负号） # not
3 * / // %
4 + -（减法）
5 ..（连接）
6 < > <= >= == ~=
7 and
8（最低） or

4.7 本章小结

类别 运算符
算术 + - * / // % ^
关系 == ~= < > <= >=
逻辑 and or not
其他 ..（连接） #（长度）
赋值 =（不是运算符，是语句）

  1. 计算 ((5 + 3) * 2 ^ 3 - 10) % 7 的结果
