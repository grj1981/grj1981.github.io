编辑器界面详解

发表于 2025-05-02 分类于 Unity3D ， 教程 Waline：
本文详细讲解编辑器界面详解，涵盖Unity3D、教程等核心知识点。「Unity3D开发基础教程」系列第2篇，共20篇。适合Unity初学者。附完整代码示例与实战技巧。

2.1 Scene 视图

Scene 视图是编辑游戏世界的核心工作区。

视图控制

  • 右键拖拽：旋转视角
  • 滚轮：缩放
  • 中键拖拽：平移视角
  • 右键 + WASD：飞行模式浏览

工具栏

工具 快捷键 功能
手形工具 Q 平移场景视图
移动工具 W 移动选中的对象
旋转工具 E 旋转选中的对象
缩放工具 R 缩放选中的对象
矩形工具 T 2D/UI 对象调整
变换工具 Y 移动/旋转/缩放组合

辅助元素

  • Grid：网格辅助线，帮助对齐
  • Gizmos：组件可视化图标（灯光、相机、音源等）
  • Skybox：背景天空盒，可在设置中关闭

2.2 Game 视图

Game 视图模拟最终发布的运行效果。

分辨率设置

  • 下拉菜单选择预设分辨率（如 1920x1080）
  • 支持自定义宽高比
  • Free Aspect：自由拉伸，适合调试

控制栏

按钮 快捷键 功能
播放 Ctrl+P 进入运行模式
暂停 Ctrl+Shift+P 暂停运行
逐帧 Ctrl+Alt+P 逐帧播放（调试用）

注意：运行模式下的修改会在退出后还原！如需保留，需在运行前修改或退出后重新调整。

2.3 Hierarchy 面板

Hierarchy 以树形结构列出当前场景的所有对象。

父子关系

将对象 A 拖拽到对象 B 上，A 成为 B 的子级：

  • 子级继承父级的 Transform 变换
  • 移动父级 → 所有子级一起移动
  • 操作子级不影响父级

右键菜单

  • Create Empty：创建空对象，常用作容器
  • 3D Object / 2D Object：创建基本形状
  • Light / Audio / UI：创建功能对象

2.4 Project 面板

Project 面板管理项目的所有资源文件。

资源管理模式

  • Two Column Layout（推荐）：左侧文件夹树 + 右侧文件列表
  • One Column Layout：单列列表模式

常用操作

  • 拖拽导入：将外部文件（FBX、PNG、MP3 等）拖入 Project 面板
  • 右键创建：创建脚本、材质、动画、预制体等
  • 搜索过滤：按类型（t:）、标签（l:）过滤资源
  • 包管理：通过 Package Manager 安装/移除功能包

资源元文件

每个资源旁都伴随一个 .meta 文件，存储 Unity 的导入设置。删除 .meta 文件会导致引用丢失。

2.5 Inspector 面板

Inspector 显示选中对象的详细属性。

组件编辑

  • 每个组件有独立的 UI 区域
  • 复选框：启用/禁用组件
  • 齿轮菜单：组件操作（Reset、Remove、Move Up/Down）
  • Add Component 按钮：添加新组件

常用功能

  • 属性的拖拽赋值：从 Project 面板拖入资源
  • 数值的多样输入：直接输入、滑块调节、颜色拾取器
  • 预设 Preset：保存组件配置为预设，方便复用

2.6 其他重要面板

面板 快捷键 功能
Console Ctrl+Shift+C 日志输出（Debug.Log）
Animation Ctrl+6 动画编辑
Animator 窗口→动画 动画状态机
Light Explorer 窗口→照明 场景灯光总览
Profiler 窗口→分析 性能分析工具

2.7 本章小结

  • Scene 视图是编辑核心，熟悉手型/移动/旋转/缩放工具
  • Game 视图预览运行效果，播放模式修改不保留
  • Hierarchy 管理场景对象，父子关系影响变换继承
  • Project 面板管理所有资源
  • Inspector 编辑组件属性

练习题

  1. 在场景中创建一个 Cube 和 Sphere，练习 W/E/R 工具的切换使用
  2. 将 Sphere 拖拽到 Cube 下成为子对象，观察移动 Cube 时的变化
  3. 在 Project 面板中创建 Scripts、Materials、Prefabs 文件夹
  4. 尝试在 Game 视图切换不同分辨率预览
