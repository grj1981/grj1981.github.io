Unity 简介与环境搭建

发表于 2025-05-01 分类于 Unity3D ， 教程 Waline：
Unity 游戏开发入门第一课，介绍 Unity 引擎核心功能、Hub 安装与版本管理、编辑器界面布局、创建第一个 3D 场景及基本操作，零基础开启 Unity 游戏开发之旅。

1.1 什么是 Unity

Unity 是一款跨平台的实时 3D 开发引擎，由 Unity Technologies 于 2005 年发布。它被广泛用于游戏开发、影视动画、建筑可视化、工业仿真等领域。

Unity 的核心优势

  • 跨平台发布：一次开发，可发布到 Windows、macOS、Linux、iOS、Android、WebGL、游戏主机等 20+ 平台
  • 可视化编辑器：所见即所得的场景编辑，拖拽式工作流
  • 组件化架构：GameObject + Component 的灵活组合模式
  • 强大的资源商店：丰富的素材、插件、工具集
  • 完善的社区生态：海量教程、文档、开源项目
  • C# 脚本语言：类型安全、性能优秀、学习曲线平缓

Unity 的应用领域

领域 代表作品
手机游戏 王者荣耀、原神、Pokémon GO
PC/主机游戏 空洞骑士、Cities: Skylines、Escape from Tarkov
VR/AR Beat Saber、Hololens 应用
影视动画 《狮子王》虚拟制作、Unity 短片《Windup》
工业/建筑 数字孪生、BIM 可视化、汽车配置器

1.2 版本选择与安装

Unity Hub（推荐）

Unity Hub 是 Unity 的项目管理工具，统一管理多个 Unity 编辑器版本和项目。

安装步骤：

  1. 访问 Unity 官网 下载 Unity Hub
  2. 安装 Unity Hub 后，在 “Installs” 标签页添加编辑器版本
  3. 推荐选择 LTS（长期支持） 版本，稳定性最佳
  4. 在安装时勾选需要的模块（如 Android Build Support、WebGL 等）

版本选择建议

用途 推荐版本
初学者学习 最新 LTS（如 2022.3 LTS）
商业项目 LTS 版本
体验最新功能 最新正式版（Tech Stream）

激活许可证

  • Unity Personal：免费，年收入低于 20 万美元
  • Unity Plus/Pro：付费，包含更多企业级功能

1.3 创建第一个项目

新建项目

  1. 打开 Unity Hub，点击 “New Project”
  2. 选择模板：
    • 2D Core：2D 游戏开发
    • 3D Core：3D 游戏开发
    • Universal 3D：使用 URP 渲染管线
  3. 设置项目名称和保存路径
  4. 点击 “Create project” 等待初始化

项目目录结构

Assets 目录是开发中主要操作的文件夹，所有导入的资源、编写的脚本、创建的场景都存放在这里。

1.4 编辑器界面概览

首次打开 Unity 编辑器，你会看到以下核心面板：

面板 功能
Scene 场景视图，可视化编辑游戏世界
Game 游戏视图，预览实际运行效果
Hierarchy 层级面板，当前场景的所有对象列表
Project 项目面板，Assets 目录的文件浏览器
Inspector 检视面板，选中对象的属性编辑

1.5 本章小结

  • Unity 是一款跨平台的实时 3D 开发引擎，应用广泛
  • 使用 Unity Hub 管理版本和项目是最佳实践
  • Assets 目录是开发的核心工作目录
  • 五大核心面板构成 Unity 编辑器的主要工作流

练习题

  1. 下载安装 Unity Hub 和一个 LTS 版本编辑器
  2. 创建一个 3D Core 项目，浏览编辑器各大面板
  3. 在 Project 面板中创建几个文件夹（Scripts、Scenes、Materials、Textures）
  4. 保存当前场景为 MainScene
